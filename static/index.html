Generate a flat-color segmented image from your 3D model using the Pixmesh pipeline. Use the output as the guidance map in the Full + 2D Map tab.

Method: Pixmesh 2D Render

Renders view(s) → VLM identifies parts → assigns Kelly palette colors → image-gen flood-fills → outputs flat-color PNG.

Renders front/back/left/right for the VLM describe step. The selected views below determine which are rendered for the describe context. Final output is always a single clean view from transforms.json.
